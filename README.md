# SRS Horse Care Website

A responsive static website for **SRS Horse Care**, featuring service information, a horse image catalog, and an appointment request interface.

## Overview

The SRS Horse Care Website is a front-end web project developed to showcase horse care services and provide visitors with information about the organization. The website includes service descriptions, a catalog gallery, and an appointment request form.

This project was built using HTML, CSS, and JavaScript and can be run locally without any additional installation.

## Features

* Responsive user-friendly design
* Home page introducing SRS Horse Care
* About page with company information
* Services page showcasing horse care services
* Catalog page with an image slider/gallery
* Appointment request form with confirmation popup
* Privacy Policy page
* Terms of Service page
* Shared navigation bar and footer across all pages

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/REAM-02/srs-horse-care-website.git
```

### Run the Project

1. Open the project folder.
2. Navigate to the `website` directory.
3. Open `index.html` in your preferred web browser.

No installation or server configuration is required.
